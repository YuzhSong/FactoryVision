# Smart Factory Vision Project

## Overview

智安工厂实时视频分析监测系统面向工厂安全生产场景，采用前后端分离与 AI 独立服务架构。

## Scope

- 当前阶段：项目骨架搭建
- 前端仅通过后端 API 获取业务数据
- AI 服务仅通过后端 API 上报检测结果
- 暂不实现复杂业务逻辑和真实模型推理

## Principles

- Keep services loosely coupled
- Prefer clear module ownership
- Drive future feature work with OpenSpec requirements first
