# 图片目录

本目录用于存放项目文档相关图片。

## 存放内容

- 架构图
- 流程图
- 数据流图
- 页面截图
- 演示截图
- 测试截图
- 部署和 Jenkins 截图

## 命名规则

建议使用清晰、稳定的英文文件名：

```text
architecture-overview.png
monitor-page-screenshot.png
jenkins-build-success.png
```

## 使用规则

1. 图片应服务于文档说明，不存放无关素材。
2. 文档引用图片时使用相对路径。
3. 页面截图应标明截图日期或对应版本。
4. 架构图若可用 Mermaid 表达，优先写在 Markdown 中；需要导出或展示截图时再放入本目录。
